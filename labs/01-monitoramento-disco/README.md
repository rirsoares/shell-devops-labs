# Lab 01 — Monitoramento de Disco em Shell Script

## Objetivo
Monitorar o uso de disco de uma partição Linux e gerar alerta quando o uso ultrapassar um limite definido.

## Conceitos abordados
- Shell Script profissional
- set -euo pipefail
- Validação de dependências
- Parsing de comandos (`df`, `awk`, `sed`)
- Códigos de saída para automação

## Como usar

```bash
chmod +x monitorar-disco.sh
./monitorar-disco.sh
